# reasonligo-vscode README

I noticed there was a syntax highlighter for PascalLIGO but there wasn't one available for ReasonLIGO which was what I was using. Therefore I started this Visual Studio Code Extension.

## Features
Syntax Highlighting
.religo .rligo Extension Recognition
